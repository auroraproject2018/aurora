特性

* 使用 和比特币一样的Secp256k1曲线,无法攻克的安全堡垒
* 未来将提供和google allo产品、whatsapp相同安全等级的 X25519曲线
* 安全协议使用TLS1.2 ,密钥交换使用椭圆曲线secp256k1,加密套件使用AES256-GCM-SHA384 
* ios 安卓 平台 使用 椭圆曲线prime256v1
